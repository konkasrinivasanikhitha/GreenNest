# GreenNest

A responsive front-end prototype for an online nursery and gardening-services marketplace.

## Run locally

Open `index.html` in a browser. No build step or dependency installation is required.

## Included interactions

- Product category filtering
- Shopping bag with add/remove and checkout confirmation
- Gardener booking form
- Newsletter signup feedback

## Next production steps

Connect authentication, inventory, booking availability, payments, and an admin dashboard to a secure backend with role-based access control.
